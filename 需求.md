# 项目需求文档

## 原始需求
STM32C8T6最小系统板，板子上有一个低电平有效的LED灯，需要这颗LED灯产生呼吸灯的效果，呼吸周期8s，要求呼吸流畅有一定视觉观赏性。

| 传感器类型 | 无 |

| OLED显示 | 否 |

| 恢复机制 | 不涉及 |
| 模式要求 | 自动 |

### 呼吸灯参数
| 参数 | 数值 | 说明 |
|------|------|------|
| 呼吸周期 | 8秒 | 暗→亮→暗一个完整周期 |
| 占空比范围 | 0%~100% | 低电平有效，PWM控制 |
| 刷新频率 | ≥100Hz | 避免肉眼可见闪烁 |
| 呼吸曲线 | 类正弦波 | 实现平滑的视觉过渡效果 |

## 实现建议

1. **PWM配置**：使用定时器输出PWM控制LED亮度
2. **呼吸算法**：采用正弦表或指数曲线实现平滑亮度变化
3. **低电平有效**：占空比0%为全亮，100%为全灭（需注意反相）
4. **周期控制**：8秒周期 = 8000ms，根据PWM分辨率计算步进间隔
