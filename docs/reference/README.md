# Reference Materials

从 datasheet / 参考代码中提取的**平台无关关键信息**，供换平台时快速复用。

**用途**：
- 不用重新翻几百页 PDF
- 换 MCU 时直接参考这些精炼参数
- AI 生成新平台代码时的知识输入

**结构**：

| 文件 | 内容 |
|------|------|
| `gpio.md` | GPIO 引脚定义、端口映射、电平极性 |
| `timer.md` | 定时器参数、中断频率、时钟配置 |
| `uart.md` | 串口波特率、引脚、缓冲区策略 |
| `led.md` | LED 硬件参数（引脚、极性、驱动方式） |
| `platform-notes.md` | 平台迁移通用注意事项、差异速查 |

**更新规则**：
- 新增外设 → 新增对应 `.md`
- 参数变更 → 同步更新对应文件
- 发现 datasheet 关键信息 → 补充到对应文件
